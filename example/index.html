break.js

The following breakpoints are set:

addBreakpoint('small', '(max-width: 600px)')
addBreakpoint('medium', '(max-width: 900px)')
addBreakpoint('large', '(max-width: 1200px)')

Which causes events to be fired on the window object: 

$(window).on('exit.small', function () {
  alert('Exiting small breakpoint')
})

$(window).on('enter.small', function () {
  alert('Entering small breakpoint')
})

// Etc...

Look in the console to see the results.
